About
Application Introduction:

The Stock Analysis app is designed with the purpose of assiting traders in identifying and designing the optimum strategy while investing in the stock market (currently Indian NSE lists are available for analysis).
The App algorithm is using time history data of the stock, and returns the information about various events of the optimum (for maximum profit on investment) trade. The stock history is fetched from the Yahoo Finance API based on the user input data (i.e. history duation, history frequency). If the result plot is showing empty, then the corresponding data at the input frequency is not available at Yahoo Finance API, so change to other frequency (preferably 'one days').


The user interface is facilitated with the following:

  • Form, to input options for the chosen stock. The options (the last 2 options in the form) for no of days of monotonic (without fluctuations) increase/decrease in stock value will guide the Python server in deciding when to purchase/sell the stock (please ensure these days value is less than the stock history range). The correlation and partial-correlation plots (available through the 'help' button in the interface at 'Home') of the stock history will assist in the initial choice for these 2 inputs. (ACF, PACF gives the correlation between the stock values at different time periods (see refereces for details), which guides the user in deciding the input options for min no of days of monotonic increase/decrease in stock price).
  • Stock Display Plot, to give prior information of the stock trajectory for the selected duration. This display plot is generated with 'Plotly' to have an interactive user experience with stock data, by exploring their Open, Close, High, Low values from the history. These interactive UI option is accessble from the top of the Display plot.
  • Stock Analysis Plot, to present the information on various trading events (optimised by the algorithm), like, stock purchase time, stock hold duration, stock sale time. The analysis is done using the stock opening value.
  • Trading Information Table, to present the details of all the trading events for the selected period.
  • UI features, to enhance the user experience,
    • Resize-Move button to move or resize the cotainers within thier boundaries,
    • Zoom button to assist the user in close observtion (through full screen mode or zoom box mode) of the Analysed plots, and
    • Help button to assist the user in chosing the input options using the correlation and auto-correlation graphs of the selected stock history.
Algorithm Introduction:

Evaluating Stock From History

The algorithm works by the time series analysis of the selected stock and evaluating the trend in correspondence with the input choices for returning the maximum profit. While doing so, the algorithm does the maximum number of purchases or (all the stock sales) that are possible based on the provided initial fund.
